# 删除回复空行

AstrBot 插件：在机器人消息发送前自动删除回复文本中的空行。(Codex编写)

## 功能

- 使用 `on_decorating_result` 钩子处理最终发送结果，不改 LLM 请求和上下文。
- 只处理 `Plain` 文本消息段，不影响图片、语音、文件、At 等组件。
- 默认将 2 个及以上连续换行压缩为 1 个换行。

例如：

```text
没关系的

我刚刚还以为自己真的漏掉了什么重要的事……
```

会变成：

```text
没关系的
我刚刚还以为自己真的漏掉了什么重要的事……
```

## 配置

`max_consecutive_newlines` 控制最多保留多少个连续换行：

- `1`：删除空行，推荐默认值。
- `2`：最多保留一个空行。
- `0`：删除所有换行，不推荐，可能让多行文本挤成一行。

## 安装

将本目录放入 AstrBot 的插件目录后，在 AstrBot 管理面板启用插件即可。
